# Next.js로 만든 Todo List Webapp

### 구성

#### 1. Next.js
#### 2. TypeScript
#### 3. 'CSS-in-JS' Styled-component
#### 4. Font
- 'Google' NotoSans & NotoSansKr
#### 5. Icons
- from iconmonstr
- modified by Figma

### API with Axios

#### /api/todos
- GET: Todo List (json)
- POST: Save new Todo

#### /api/todos/[id].ts
- PATCH: update status (checked/unchecked)
- DELETE: delete todo
